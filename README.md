## What is ZineOS?

ZineOS is an experimental web app you can use to make your own zines (“DIY magazines”) right in your browser. Everything happens in one HTML file: no build tools, backend, or install required.

**Features:**
- Create multi-page zines in a visual drag-and-drop editor.
- Add headlines, body text, images, GIFs, videos, and audio players.
- Layer content like Photoshop, use custom fonts, and set backgrounds.
- Export as standalone HTML (to share) or re-editable `.zine` file.
- Store your zines locally, import/export, and manage a zine library.

## Getting Started

- Open [`zine3.html`](./zine3.html)
- Click “Create New Zine”
- Drag in headlines, images, videos, audio, GIFs, etc.
- Export your zine as HTML or `.zine` file  
- Import/export as you please!

---

## Export/Import

- **Export:** Use the “Export” button in the editor. Choose HTML (share/read-only) or `.zine` (future editing).
- **Import:** Click “Import” in the library view and upload a `.zine` file.
