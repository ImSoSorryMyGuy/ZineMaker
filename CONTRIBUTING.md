# Contributing to ZineOS

Thanks for your interest in helping ZineOS! All collaboration, feedback, bug reports, or code contributions are welcome.

## How to Contribute

- **Bug Reports & Feature Requests:**  
  Use [Issues](https://github.com/ImSoSorryMyGuy/ZineMaker/issues) to report a bug, propose a new feature, or ask questions.

- **Pull Requests:**  
  Fork the repo, make your changes in a branch, and submit a Pull Request.  
  Please add a short description of your changes and reference any related issues.

- **Code Review:**  
  Feel free to critique, share improvements, restructure code, or point out things you’d do differently.

## Coding Style

- Keep it simple—this project is meant for learning and remixing.
- No build tools or frameworks (for now); everything is in a single HTML file.
- Leave comments in your code if you make big changes.
- Please test in multiple browsers if possible!

## Community

Be considerate and constructive. All are welcome to participate, ask questions, or just say hi!
